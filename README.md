# CSB Client

Fork of Meteor Client made for CursedSkyblock.net.
Join our [Discord](https://discord.gg/YaZVgazkYn).

### Installation
Follow guide at meteorclient.com.

### Building
- Clone this repository
- Run `./gradlew build`
