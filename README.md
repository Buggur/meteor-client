# CSB Client

Fork of Meteor Client made for CursedSkyblock.net.
Join our [Discord](https://discord.gg/gvV6YshfCx).

## Installation
Follow guide at [meteorclient.com](https://meteorclient.com/info).

## Building
- Clone this repository
- Run `./gradlew build`
